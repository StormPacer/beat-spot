What is Beat Spot?

Beat Spot is a tool that converts a Spotify playlist into a Beat Saber playlist. It also has some features like downloading all maps of a certain type (see "Beat Spot Functions").

API LIMITS
Due to the Spotify's API limiting, you are allowed to use the Spotify to playlist feature 3 times per 5 minutes.

Going through all the songs that the tool can go through takes a minute, so please be patient!

NOTE
This is not a perfect tool, and I don't claim for it to be.

The tool will take only the first 200 songs of the playlist!
